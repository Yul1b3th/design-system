Data Table

El componente de tabla de datos (ds-data-table) se utiliza para mostrar datos tabulares de manera organizada y accesible. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-data-table [data]="data" [columns]="columns"></ds-data-table>

Configuración

El componente de tabla de datos acepta las siguientes propiedades de entrada (@Input):

  • data (Array<{ [key: string]: any }>): Lista de objetos que representan las filas de la tabla.
  • columns (Array<{ header: string, field: string }>): Lista de objetos que representan las columnas de la tabla. Cada objeto debe tener una etiqueta (header) y una clave (field) que corresponde a las propiedades de los objetos en data.

Accesibilidad

El componente de tabla de datos incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de tabla de datos es una herramienta esencial para mostrar datos tabulares de manera organizada y accesible. Con sus múltiples opciones de configuración y soporte para accesibilidad, se puede adaptar a una amplia variedad de casos de uso en la aplicación.
