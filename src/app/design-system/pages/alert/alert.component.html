Alert

El componente de alerta (ds-alert) se utiliza para mostrar mensajes importantes a los usuarios. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-alert
  type="success"
  message="This is a success alert!"
  [dismissible]="true"
></ds-alert>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
type 'success' | 'error' | 'warning' | 'info' Tipo de alerta a mostrar. "success"
message string Mensaje a mostrar en la alerta. "This is a success alert!"
dismissible boolean Define si la alerta puede ser cerrada por el usuario. true

Accesibilidad

El componente de alerta incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de alerta es una herramienta útil para mostrar mensajes importantes a los usuarios. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
