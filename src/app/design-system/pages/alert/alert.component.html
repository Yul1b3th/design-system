Alertas

El componente de alerta (ds-alert) se utiliza para mostrar mensajes importantes a los usuarios. Este componente es altamente personalizable y soporta múltiples variantes para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-alert type="success">¡Operación exitosa!</ds-alert>
<ds-alert type="danger">¡Error! Algo salió mal.</ds-alert>
<ds-alert type="warning">Advertencia: Por favor, revise los datos.</ds-alert>
<ds-alert type="info">Información: Este es un mensaje informativo.</ds-alert>

Configuración

El componente de alerta acepta las siguientes propiedades de entrada (@Input):

  • type (string): Tipo de alerta. Puede ser success, danger, warning o info.
  • dismissible (boolean): Indica si la alerta puede ser cerrada por el usuario. Valor por defecto: false.

Accesibilidad

El componente de alerta incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de alerta es una herramienta esencial para comunicar mensajes importantes a los usuarios. Con sus múltiples variantes y opciones de configuración, se puede adaptar a una amplia variedad de casos de uso en la aplicación.
