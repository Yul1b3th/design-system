Spinner

El componente de spinner (ds-spinner) se utiliza para indicar que una operación está en curso. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:


        <ds-spinner
          [size]="'medium'"
          [color]="'primary'"
        ></ds-spinner>

Configuración

El componente de spinner acepta las siguientes propiedades de entrada (@Input):

  • size ('small' | 'medium' | 'large'): Tamaño del spinner. Valor por defecto: 'medium'.
  • color ('primary' | 'secondary' | 'tertiary'): Color del spinner. Valor por defecto: 'primary'.

Accesibilidad

El componente de spinner incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.
