Dropdown

El componente de menú desplegable (ds-dropdown) se utiliza para mostrar una lista de opciones que el usuario puede seleccionar. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-dropdown
  [label]="'Select an option'"
  [options]="[
    { label: 'Option 1', value: 1 },
    { label: 'Option 2', value: 2 },
    { label: 'Option 3', value: 3 }
  ]"
></ds-dropdown>

Configuración

El componente de menú desplegable acepta las siguientes propiedades de entrada (@Input):

  • label (string): Texto que se mostrará en el botón del menú desplegable.
  • options (Array<{ label: string, value: any }>): Lista de opciones que se mostrarán en el menú desplegable. Cada opción debe tener una etiqueta (label) y un valor (value).
  • selectedValue (any): Valor de la opción seleccionada.
  • disabled (boolean): Indica si el menú desplegable está deshabilitado. Valor por defecto: false.

Accesibilidad

El componente de menú desplegable incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de menú desplegable es una herramienta útil para permitir a los usuarios seleccionar opciones de una lista. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
