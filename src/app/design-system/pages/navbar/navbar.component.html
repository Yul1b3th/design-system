Navbar

El componente de barra de navegación (ds-navbar) se utiliza para proporcionar una navegación consistente en la parte superior de la aplicación. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-navbar>
  <ds-navbar-brand>Brand</ds-navbar-brand>
  <ds-navbar-nav>
    <ds-navbar-item><a href="#">Home</a></ds-navbar-item>
    <ds-navbar-item><a href="#">About</a></ds-navbar-item>
    <ds-navbar-item><a href="#">Contact</a></ds-navbar-item>
  </ds-navbar-nav>
</ds-navbar>

Configuración

El componente de barra de navegación acepta las siguientes propiedades de entrada (@Input):

  • brand (string): El nombre de la marca o logo que se mostrará en la barra de navegación.
  • items (Array<{ label: string, url: string }>): Lista de elementos de navegación. Cada elemento debe tener una etiqueta (label) y una URL (url).

Accesibilidad

El componente de barra de navegación incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de barra de navegación es una herramienta esencial para proporcionar una navegación consistente en la parte superior de la aplicación. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
