Card

El componente de tarjeta (ds-card) se utiliza para mostrar contenido en un formato de tarjeta. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-card
  imageUrl="https://ezhub.vercel.app/assets/img/bedrooms/1.jpg"
  title="Card Title"
  body="This is a card body text."
  buttonText="Learn More"
  buttonUrl="#"
></ds-card>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
imageUrl string URL de la imagen a mostrar en la parte superior de la tarjeta. "https://ezhub.vercel.app/assets/img/bedrooms/1.jpg"
title string Título de la tarjeta. "Card Title"
body string Texto del cuerpo de la tarjeta. "This is a card body text."
buttonText string Texto del botón en la tarjeta. "Learn More"
buttonUrl string URL al que redirige el botón de la tarjeta. "#"

Accesibilidad

El componente de tarjeta incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de tarjeta es una herramienta útil para mostrar contenido en un formato de tarjeta. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
