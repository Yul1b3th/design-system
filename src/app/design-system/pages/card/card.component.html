Cards

El componente de tarjeta (ds-card) es un contenedor flexible que permite mostrar contenido variado de manera organizada y atractiva. Este componente es altamente personalizable y soporta múltiples variantes para adaptarse a diferentes contextos de uso.

Variantes de Tarjeta

El componente de tarjeta soporta varias variantes que se pueden aplicar mediante clases CSS. Las variantes disponibles son:

  • Default: Tarjeta predeterminada utilizada para mostrar contenido general.
  • Dark Mode: Tarjeta con esquema de colores oscuros para entornos de baja luminosidad.
  • Light Mode: Tarjeta con esquema de colores claros para entornos de alta luminosidad.

Uso

Ejemplo de uso en HTML:

<ds-card>
  <img card-image src="https://ezhub.vercel.app/assets/img/bedrooms/1.jpg" alt="Sample Room" class="img-responsive" />
  <h5 card-title>Sample Room</h5>
  <p card-text>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
  <p card-price>100 <span class="currency">€</span></p>
  <ds-button card-button btnClass="btn-secondary">See more</ds-button>
</ds-card>
