Card Skeleton

El componente ds-card-skeleton se utiliza para mostrar un skeleton de carga para las cards. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de implementación en HTML:


      <ds-card-skeleton>
        <div card-image></div>
        <div card-title></div>
        <div card-body></div>
        <div card-button></div>
      </ds-card-skeleton>
