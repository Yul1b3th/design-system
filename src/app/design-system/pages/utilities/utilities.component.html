Utilidades

En esta sección, exploraremos las utilidades disponibles en el sistema de diseño. Estas utilidades permiten aplicar estilos comunes de manera rápida y sencilla.

Colores

Las utilidades de colores permiten aplicar rápidamente colores de texto, fondo y borde utilizando las variables de colores definidas. 


      @each $color-name, $color-value in $colors {
        .text-#{$color-name} {
          color: get-color($color-name);
        }

        .bg-#{$color-name} {
          background-color: get-color($color-name);
        }

        .border-#{$color-name} {
          border-color: get-color($color-name);
        }
      }

Espaciado

Las utilidades de espaciado permiten aplicar rápidamente márgenes y rellenos utilizando las variables de espaciado definidas. 


      @each $size-name, $size-value in $spacing {
        .m-#{$size-name} {
          margin: get-spacing($size-name);
        }

        .mt-#{$size-name} {
          margin-top: get-spacing($size-name);
        }

        .mr-#{$size-name} {
          margin-right: get-spacing($size-name);
        }

        .mb-#{$size-name} {
          margin-bottom: get-spacing($size-name);
        }

        .ml-#{$size-name} {
          margin-left: get-spacing($size-name);
        }

        .mx-#{$size-name} {
          margin-left: get-spacing($size-name);
          margin-right: get-spacing($size-name);
        }

        .my-#{$size-name} {
          margin-top: get-spacing($size-name);
          margin-bottom: get-spacing($size-name);
        }

        .p-#{$size-name} {
          padding: get-spacing($size-name);
        }

        .pt-#{$size-name} {
          padding-top: get-spacing($size-name);
        }

        .pr-#{$size-name} {
          padding-right: get-spacing($size-name);
        }

        .pb-#{$size-name} {
          padding-bottom: get-spacing($size-name);
        }

        .pl-#{$size-name} {
          padding-left: get-spacing($size-name);
        }

        .px-#{$size-name} {
          padding-left: get-spacing($size-name);
          padding-right: get-spacing($size-name);
        }

        .py-#{$size-name} {
          padding-top: get-spacing($size-name);
          padding-bottom: get-spacing($size-name);
        }
      }

Texto

Las utilidades de texto permiten aplicar rápidamente estilos de alineación, truncamiento de texto y peso de fuente. 


  // Mixin para truncar texto
  @mixin apply-line-clamp($lines) {
    @if $lines > 0 {
      display: -webkit-box;
      -webkit-box-orient: vertical;
      -webkit-line-clamp: $lines;
      overflow: hidden;
      text-overflow: ellipsis;
    } @else {
      @error "El mixin `apply-line-clamp` requiere un número mayor a 0 para `$lines`.";
    }
  }

  // Clase global para truncar texto
  .line-clamp-1 {
    @include apply-line-clamp(1);
  }

  .line-clamp-2 {
    @include apply-line-clamp(2);
  }

  .line-clamp-3 {
    @include apply-line-clamp(3);
  }

  .line-clamp-4 {
    @include apply-line-clamp(4);
  }

  .line-clamp-5 {
    @include apply-line-clamp(5);
  }

  // Text alignment
  .text-center {
    text-align: center;
  }

  .text-left {
    text-align: left;
  }

  .text-right {
    text-align: right;
  }

  // Font weight
  .fw-semibold {
    font-weight: 600;
  }

  .fw-bold {
    font-weight: 700;
  }

  .fw-normal {
    font-weight: 400;
  }

  // Font size
  @each $size-name, $size-value in $font-sizes {
    .fs-#{$size-name} {
      font-size: get-font-size($size-name);
    }
  }

Utilidades Generales

Las utilidades generales incluyen clases para aplicar estilos comunes como imágenes responsivas. 


      // Clase para imágenes responsivas
      .img-responsive {
        width: 100%;
        height: 100%;
        object-position: center;
        object-fit: cover;
      }

Visibilidad

Las utilidades de visibilidad permiten controlar rápidamente la visibilidad de los elementos. 


      .hidden {
        display: none !important;
      }

      .visible {
        display: block !important;
      }

      .invisible {
        visibility: hidden !important;
      }

      .d-block {
        display: block !important;
      }

      .d-inline {
        display: inline !important;
      }

      .d-inline-block {
        display: inline-block !important;
      }

      .d-flex {
        display: flex !important;
      }

      .d-none {
        display: none !important;
      }
