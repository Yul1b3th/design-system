Buttons

El componente ds-button se utiliza para crear botones interactivos con diversas opciones de personalización. Es completamente accesible y adecuado para formularios o acciones independientes.

Uso

Ejemplo de uso en HTML:

<ds-button btnClass="btn-primary">Primary</ds-button>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
type string Define el tipo de botón. Puede ser 'button', 'submit' o 'reset'. "button"
btnClass string Clase CSS para el botón. "btn-primary"
size string Define el tamaño del botón. Puede ser 'btn-sm', 'btn-md' o 'btn-lg'. "btn-md"
block boolean Si es verdadero, el botón ocupará todo el ancho del contenedor. false
outline boolean Si es verdadero, el botón tendrá un estilo de contorno. false
disabled boolean Desactiva el botón. false
ariaLabel string Etiqueta aria para accesibilidad. ""
iconSvg string SVG del icono a mostrar dentro del botón. <svg ...></svg>
bgColor string Color de fondo personalizado para el botón. "#ff5733"
textColor string Color de texto personalizado para el botón. "#ffffff"
borderRadius string Radio de borde personalizado para el botón. "8px"

Accesibilidad

Se debe asegurar de proporcionar una etiqueta aria-label para describir la acción del botón cuando el texto del botón no es suficiente para describir su propósito.

Variaciones

El componente de botón soporta varias variantes que se pueden aplicar mediante clases CSS. Las variantes disponibles son:

  • Primary Buttons
  • Secondary Buttons
  • Success Buttons
  • Danger Buttons
  • Warning Buttons
  • Info Buttons
  • Light Buttons
  • Dark Buttons
  • With Icon
  • Custom Buttons

Conclusión

El componente de botón es una parte esencial de cualquier sistema de diseño. Con las variantes y tamaños disponibles, se pueden crear botones que se adapten a cualquier contexto de uso en la aplicación, asegurando una experiencia de usuario consistente y accesible.
