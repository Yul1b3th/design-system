Botones

El componente de botón (ds-button) es un elemento interactivo que permite a los usuarios realizar acciones en la interfaz. Este componente es altamente personalizable y soporta múltiples variantes y tamaños para adaptarse a diferentes contextos de uso.

Variantes de Botón

El componente de botón soporta varias variantes que se pueden aplicar mediante clases CSS. Las variantes disponibles son:

  • Primary: Botón principal utilizado para acciones primarias.
  • Secondary: Botón secundario utilizado para acciones secundarias.
  • Success: Botón utilizado para indicar acciones exitosas.
  • Danger: Botón utilizado para acciones peligrosas o destructivas.
  • Warning: Botón utilizado para advertencias.
  • Info: Botón utilizado para información adicional.
  • Light: Botón con estilo claro.
  • Dark: Botón con estilo oscuro.

Tamaños de Botón

El componente de botón también soporta diferentes tamaños:

  • Small (btn-sm): Botón pequeño.
  • Medium (btn-md): Botón mediano (tamaño predeterminado).
  • Large (btn-lg): Botón grande.

Uso

Ejemplo de uso en HTML:

<ds-button type="button" variant="primary" size="sm">Botón Primario Pequeño</ds-button>
<ds-button type="button" variant="primary" size="md">Botón Primario Mediano</ds-button>
<ds-button type="button" variant="primary" size="lg">Botón Primario Grande</ds-button>
<ds-button type="button" variant="secondary" size="sm">Botón Secundario Pequeño</ds-button>
<ds-button type="button" variant="secondary" size="md">Botón Secundario Mediano</ds-button>
<ds-button type="button" variant="secondary" size="lg">Botón Secundario Grande</ds-button>
<ds-button type="button" variant="success" size="sm">Botón de Éxito Pequeño</ds-button>
<ds-button type="button" variant="success" size="md">Botón de Éxito Mediano</ds-button>
<ds-button type="button" variant="success" size="lg">Botón de Éxito Grande</ds-button>

Accesibilidad

Se debe asegurar de proporcionar una etiqueta aria-label para describir la acción del botón cuando el texto del botón no es suficiente para describir su propósito.

Conclusión

El componente de botón es una parte esencial de cualquier sistema de diseño. Con las variantes y tamaños disponibles, se pueden crear botones que se adapten a cualquier contexto de uso en la aplicación, asegurando una experiencia de usuario consistente y accesible.
