Personalización

En esta sección, aprenderás cómo personalizar el Design System utilizando variables, funciones y mixins.

Variables

Las variables permiten definir valores reutilizables en todo el sistema de diseño. A continuación se presentan las principales variables utilizadas:

Tipografías


        // Tipografías
        $fonts: (
          primary: (
            "BeVietnamPro",
            "Arial, sans-serif",
          ),
          secondary: "Arial, sans-serif",
        );

Tamaños de fuente


        // Tamaños de fuente
        $font-sizes: (
          xs: 0.75rem,
          sm: 0.875rem,
          base: 1rem,
          lg: 1.125rem,
          xl: 1.25rem,
          "2xl": 1.5rem,
          "3xl": 1.75rem,
          "4xl": 2rem,
          "5xl": 2.5rem,
        );

Alturas de línea


        // Alturas de línea
        $line-heights: (
          xs: 1rem,
          sm: 1.25rem,
          base: 1.5rem,
          lg: 1.75rem,
          xl: 2rem,
        );

Espaciados


        // Espaciados
        $spacing: (
          xs: 0.25rem,
          sm: 0.5rem,
          md: 1rem,
          lg: 2rem,
          xl: 4rem,
        );

        // Mapas de colores expandidos
        $backgrounds: map.merge(

Los mapas de colores expandidos permiten definir colores de fondo predeterminados e invertidos, lo que facilita la implementación de temas claros y oscuros en el sistema de diseño.

Funciones

Las funciones en Sass permiten reutilizar lógica y cálculos en todo el sistema de diseño. A continuación se presentan las principales funciones utilizadas:

Función genérica para obtener valores de un mapa

      
        // Función genérica para obtener valores de un mapa
        @function get-map-value($map, $key) {
          @if map.has-key($map, $key) {
            @return map.get($map, $key);
          } @else {
            @error "La clave {{$key}} no existe en el mapa proporcionado.";
          }
        }

Esta función permite obtener valores de un mapa dado una clave específica. Es útil para acceder a valores definidos en mapas como colores, tamaños de fuente, etc.

Función para obtener una fuente por nombre

      
        // Función para obtener una fuente por nombre
        @function get-font($font-name) {
          @return get-map-value($fonts, $font-name);
        }

Esta función devuelve la fuente correspondiente a un nombre de fuente específico.

Función para obtener un tamaño de fuente

      
        // Función para obtener un tamaño de fuente
        @function get-font-size($size-name) {
          @return get-map-value($font-sizes, $size-name);
        }

Esta función devuelve el tamaño de fuente correspondiente a un nombre de tamaño específico.

Función para obtener una altura de línea

      
        // Función para obtener una altura de línea
        @function get-line-height($size-name) {
          @return get-map-value($line-heights, $size-name);
        }

Esta función devuelve la altura de línea correspondiente a un nombre de tamaño específico.

Función para obtener un espaciado

      
        // Función para obtener un espaciado
        @function get-spacing($size-name) {
          @return get-map-value($spacing, $size-name);
        }

Esta función devuelve el valor de espaciado correspondiente a un nombre de tamaño específico.

Función para obtener un color

      
        // Función para obtener un color
        @function get-color($color-name...) {
          $color-map: $colors;

          @each $name in $color-name {
            @if map.has-key($color-map, $name) {
              $color-map: map.get($color-map, $name);
            } @else {
              @error "La clave {{$name}} no existe en el mapa de colores.";
            }
          }

          @if meta.type-of($color-map) == "map" {
            @error "El valor {{$color-name}} no es un color válido.";
          }

          @return $color-map;
        }

Esta función devuelve el valor del color correspondiente a un nombre de color específico.

Función para obtener un color de fondo

      
        // Función para obtener un color de fondo
        @function get-background-color($background-name...) {
          $color-map: $backgrounds;

          @each $name in $background-name {
            @if map.has-key($color-map, $name) {
              $color-map: map.get($color-map, $name);
            } @else {
              @error "La clave {{$name}} no existe en el mapa de colores de fondo.";
            }
          }

          @if meta.type-of($color-map) != "color" {
            @error "El valor {{$background-name}} no es un color válido.";
          }

          @return $color-map;
        }

Esta función devuelve el valor del color de fondo correspondiente a un nombre de fondo específico.

Función para convertir colores a HSL

      
        // Función para convertir colores a HSL
        @function color-to-hsl($color) {
          @return hsl(
            color.channel($color, "hue", $space: hsl),
            color.channel($color, "saturation", $space: hsl),
            color.channel($color, "lightness", $space: hsl)
          );
        }

Esta función convierte un color a su representación en HSL (Hue, Saturation, Lightness).

Función para convertir colores a HSLA con transparencia

      
        // Función para convertir colores a HSLA con transparencia
        @function color-to-hsla($color, $alpha) {
          @return hsla(
            color.channel($color, "hue", $space: hsl),
            color.channel($color, "saturation", $space: hsl),
            color.channel($color, "lightness", $space: hsl),
            $alpha
          );
        }

Esta función convierte un color a su representación en HSLA (Hue, Saturation, Lightness, Alpha) con un valor de transparencia.

Función para obtener un breakpoint

      
        // Función para obtener un breakpoint
        @function get-breakpoint($size-name) {
          @return get-map-value($breakpoints, $size-name);
        }

Esta función devuelve el valor del breakpoint correspondiente a un nombre de tamaño específico.

Mixins

Los mixins en Sass permiten reutilizar bloques de código CSS en todo el sistema de diseño. A continuación se presentan los principales mixins utilizados:

Mixins relacionados con tipografía

      
        // Establece la familia tipográfica
        @mixin apply-font-family($font: primary) {
          font-family: get-font($font);
        }

        // Aplica propiedades tipográficas: tamaño, altura de línea, peso, margen inferior, y color
        @mixin apply-typography(
          $font-size,
          $line-height: medium,
          $font-weight: normal,
          $margin-bottom: null,
          $text-color: null
        ) {
          font-size: get-font-size($font-size);
          line-height: get-line-height($line-height);
          font-weight: $font-weight;

          @if $margin-bottom != null {
            margin-bottom: get-spacing($margin-bottom);
          }
          @if $text-color != null {
            color: get-color($text-color);
          }
        }

Mixins relacionados con espaciado

      
        // Aplica márgenes o rellenos
        @mixin apply-spacing($property, $size) {
          {{$property}}: get-spacing($size);
        }

        // Aplica márgenes y rellenos de forma abreviada
        @mixin apply-spacing-shorthand($type, $size) {
          {{$type}}-top: get-spacing($size);
          {{$type}}-right: get-spacing($size);
          {{$type}}-bottom: get-spacing($size);
          {{$type}}-left: get-spacing($size);
        }

Mixins relacionados con colores

      
        // Aplica colores (textos, fondos, bordes)
        @mixin apply-color($property, $color-name...) {
          {{$property}}: get-color($color-name...);
        }

        // Aplica colores específicos en modo claro y oscuro
        @mixin apply-color-mode($light-color, $dark-color) {
          color: get-color($light-color);
          @media (prefers-color-scheme: dark) {
            color: get-color($dark-color);
          }
        }

        // Aplica un color de fondo
        @mixin apply-background-color($background-name...) {
          background-color: get-background-color($background-name...);
        }

Mixins relacionados con diseño responsivo

      
        // Aplica estilos en un breakpoint específico
        @mixin apply-media-breakpoint($size) {
          @media (min-width: get-breakpoint($size)) {
            @content;
          }
        }

Mixins relacionados con diseño visual

      
        // Aplica bordes redondeados
        @mixin apply-border($width: 1px, $style: solid, $color-name...) {
          $color: get-color($color-name...);
          border: $width $style $color;
        }

        // Aplica bordes redondeados
        @mixin apply-border-radius($radius) {
          border-radius: $radius;
        }

        // Aplica una sombra
        @mixin apply-box-shadow(
          $x,
          $y,
          $blur,
          $spread: 0,
          $color: rgba(0, 0, 0, 0.25)
        ) {
          box-shadow: $x $y $blur $spread $color;
        }

        // Aplica un gradiente lineal
        @mixin apply-gradient($start-color, $end-color, $direction: to right) {
          background: linear-gradient($direction, $start-color, $end-color);
        }

        // Controla la opacidad de un color
        @mixin apply-color-opacity($color, $opacity) {
          background-color: rgba(
            color.channel($color, "red"),
            color.channel($color, "green"),
            color.channel($color, "blue"),
            $opacity
          );
        }

Mixins relacionados con Flexbox

      
        // Configura un contenedor flex
        @mixin apply-flex-container($direction: row, $justify: center, $align: center) {
          display: flex;
          flex-direction: $direction;
          justify-content: $justify;
          align-items: $align;
        }

        // Configura un ítem de flex
        @mixin apply-flex-item($grow: 0, $shrink: 1, $basis: auto) {
          flex: $grow $shrink $basis;
        }

Mixins relacionados con animación

      
        // Aplica transiciones suaves
        @mixin apply-transition(
          $properties: all,
          $duration: 0.3s,
          $timing-function: ease-in-out
        ) {
          transition: $properties $duration $timing-function;
        }
