Paginación

El componente de paginación (ds-pagination) se utiliza para dividir el contenido en páginas y permitir a los usuarios navegar entre ellas. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-pagination [totalItems]="100" [itemsPerPage]="10" [currentPage]="currentPage" (currentPageChange)="onPageChange($event)"></ds-pagination>

Configuración

El componente de paginación acepta las siguientes propiedades de entrada (@Input):

  • totalItems (number): El número total de elementos a paginar.
  • itemsPerPage (number): El número de elementos por página.
  • currentPage (number): La página actual.

Accesibilidad

El componente de paginación incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de paginación es una herramienta esencial para dividir el contenido en páginas y permitir a los usuarios navegar entre ellas. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
