Slider

El componente de control deslizante (ds-slider) se utiliza para permitir a los usuarios seleccionar un valor de un rango continuo o discreto. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-slider [(ngModel)]="sliderValue" [min]="0" [max]="100" [step]="1"></ds-slider>

Configuración

El componente de control deslizante acepta las siguientes propiedades de entrada (@Input):

  • ngModel (number): El valor seleccionado del control deslizante.
  • min (number): El valor mínimo del rango.
  • max (number): El valor máximo del rango.
  • step (number): El incremento entre valores del rango.
  • disabled (boolean): Indica si el control deslizante está deshabilitado. Valor por defecto: false.

Accesibilidad

El componente de control deslizante incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.
