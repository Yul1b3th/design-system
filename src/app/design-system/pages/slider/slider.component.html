Slider

El componente de control deslizante (ds-slider) se utiliza para permitir a los usuarios seleccionar un valor de un rango continuo o discreto. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:


        <form [formGroup]="form">
          <ds-slider
            formControlName="sliderValue"
            id="slider"
            label="Select Value"
            [min]="0"
            [max]="100"
            [step]="1"
            [ariaRequired]="true"
          ></ds-slider>
        </form>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
label string Texto mostrado como etiqueta del campo. "Seleccionar Valor"
min number Valor mínimo permitido para el control deslizante. 0
max number Valor máximo permitido para el control deslizante. 100
step number Incremento de valor permitido para el control deslizante. 1
formControlName string Vincula el componente a un control de un formulario reactivo. "sliderValue"
required boolean Define si el campo es obligatorio. true
disabled boolean Desactiva el control deslizante. false
errorMessage string Mensaje de error personalizado mostrado para errores de validación. "Este campo es obligatorio"

Accesibilidad
