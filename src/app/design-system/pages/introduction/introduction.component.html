Bienvenido al Design System

Este es un sistema de diseño sencillo creado para facilitar el desarrollo de interfaces consistentes y accesibles.

Cómo Empezar

Para comenzar a utilizar este Design System, sigue los siguientes pasos:

  1. Descarga el proyecto desde el repositorio.
  2. Instala las dependencias necesarias utilizando npm install.
  3. Explora los componentes y estilos disponibles en la documentación.

Personalización

Este Design System es altamente personalizable. Puedes ajustar las variables, funciones y mixins para adaptarlo a tus necesidades específicas.

Consulta la sección de Personalización para más detalles.

Importación de Estilos

El archivo main.scss centraliza la importación de todos los estilos y utilidades del sistema de diseño. A continuación se muestra el contenido del archivo: 


    @use "abstracts/variables" as *;
    @use "abstracts/functions" as *;
    @use "abstracts/mixins" as *;
    @use "base/reboot" as *;
    @use "base/typography" as *;
    @use "layout/grid" as *;
    @use "layout/helpers" as *;
    @use "themes/dark" as *;
    @use "themes/light" as *;
    @use "utilities/spacing" as *;
    @use "utilities/visibility" as *;
    @use "utilities/colors" as *;
    @use "utilities/text" as *;
    @use "utilities/utilities" as *;

Este archivo asegura que todos los estilos y utilidades estén disponibles en toda la aplicación.
