Carousel

El componente de carrusel (ds-carousel) permite mostrar una serie de imágenes de manera interactiva, con soporte para navegación y reproducción automática. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-carousel
  [photos]="[
    'https://ezhub.vercel.app/assets/img/bedrooms/1.jpg',
    'https://ezhub.vercel.app/assets/img/bedrooms/2.jpg',
    'https://ezhub.vercel.app/assets/img/bedrooms/3.jpg',
    'https://ezhub.vercel.app/assets/img/bedrooms/4.jpg',
    'https://ezhub.vercel.app/assets/img/bedrooms/5.jpg'
  ]"
></ds-carousel>

Configuración

El componente de carrusel acepta las siguientes propiedades de entrada (@Input):

  • photos (string[]): Lista de URLs de las imágenes a mostrar en el carrusel.

Accesibilidad

El componente de carrusel incluye soporte para accesibilidad mediante el uso de atributos aria y controles de navegación accesibles.

Conclusión

El componente de carrusel es una herramienta poderosa para mostrar contenido visual de manera interactiva y atractiva. Con sus múltiples opciones de configuración y soporte para accesibilidad, se puede adaptar a una amplia variedad de casos de uso en la aplicación.
