Date Picker

El componente ds-date-picker se utiliza para permitir a los usuarios seleccionar una fecha de manera interactiva. Este componente es altamente personalizable y soporta múltiples configuraciones.

Uso

Ejemplo de uso en HTML:


        <form [formGroup]="form">
          <ds-date-picker
            formControlName="datePicker"
            id="datePicker"
            label="Select Date"
            placeholder="Select a date"
            [ariaRequired]="true"
          ></ds-date-picker>
        </form>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
label string Texto mostrado como etiqueta del campo. "Seleccionar Fecha"
placeholder string Texto mostrado dentro del campo cuando está vacío. "Selecciona una fecha"
formControlName string Vincula el componente a un control de un formulario reactivo. "datePicker"
required boolean Define si el campo es obligatorio. true
disabled boolean Desactiva el campo de entrada. false
errorMessage string Mensaje de error personalizado mostrado para errores de validación. "Este campo es obligatorio"
minDate string Fecha mínima seleccionable. "2023-01-01"
maxDate string Fecha máxima seleccionable. "2023-12-31"
disabledDates string[] Lista de fechas que no se pueden seleccionar. ["2023-05-01", "2023-05-02"]
