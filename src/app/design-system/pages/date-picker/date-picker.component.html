Date Picker

El componente de selector de fecha (ds-date-picker) se utiliza para permitir a los usuarios seleccionar una fecha de manera interactiva. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-date-picker [(ngModel)]="selectedDate"></ds-date-picker>

Configuración

El componente de selector de fecha acepta las siguientes propiedades de entrada (@Input):

  • ngModel (Date): La fecha seleccionada.
  • minDate (Date): La fecha mínima seleccionable.
  • maxDate (Date): La fecha máxima seleccionable.
  • disabledDates (Date[]): Lista de fechas que no se pueden seleccionar.

Accesibilidad

El componente de selector de fecha incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de selector de fecha es una herramienta útil para permitir a los usuarios seleccionar fechas de manera interactiva. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
