Layout

En esta sección, se explora el sistema de diseño de layout que se utiliza para crear estructuras de página consistentes y responsivas.

Grid

El sistema de grid se basa en un diseño de 12 columnas que permite crear layouts flexibles y responsivos. A continuación se presentan las principales clases y mixins utilizados:

Contenedor


      .container {
        width: 100%;
        max-width: 100%; // Por defecto ocupa todo el ancho
        margin-left: auto;
        margin-right: auto;
        padding-left: get-spacing(xs);
        padding-right: get-spacing(xs);

        @each $breakpoint, $value in $breakpoints {
          @if $breakpoint != xs {
            // Evita el breakpoint más pequeño
            @include apply-media-breakpoint($breakpoint) {
              max-width: $value; // Ancho máximo basado en el breakpoint
            }
          }
        }
      }

La clase .container se utiliza para envolver el contenido y aplicar márgenes y rellenos automáticos, asegurando que el contenido esté centrado y tenga un ancho máximo basado en los breakpoints definidos.

Fila


      .row {
        display: flex;
        flex-wrap: wrap;
        margin-left: -get-spacing(xs);
        margin-right: -get-spacing(xs);
      }

La clase .row se utiliza para crear una fila de columnas. Utiliza flexbox para asegurar que las columnas se distribuyan correctamente y se envuelvan según sea necesario.

Columnas


      [class*="col-"] {
        padding-left: get-spacing(xs);
        padding-right: get-spacing(xs);
        flex: 0 0 auto;
        width: 100%;

        @for $i from 1 through 12 {
          &.col-#{$i} {
            width: math.div(100%, 12) * $i;
          }
        }

        @each $breakpoint, $value in $breakpoints {
          @if $breakpoint != xs {
            @include apply-media-breakpoint($breakpoint) {
              @for $i from 1 through 12 {
                &.col-#{$breakpoint}-#{$i} {
                  width: math.div(100%, 12) * $i;
                }
              }
            }
          }
        }
      }

Las clases de columna se utilizan para definir el ancho de las columnas en diferentes breakpoints. Utilizan un sistema de 12 columnas para asegurar que el contenido se distribuya de manera uniforme.

Helpers

Los helpers son clases utilitarias que facilitan la aplicación de estilos comunes y repetitivos. A continuación se presentan algunos de los helpers más utilizados:

Clearfix


      .clearfix::after {
        content: "";
        display: table;
        clear: both;
      }

La clase .clearfix se utiliza para aplicar el clearfix a un elemento, asegurando que los elementos flotantes dentro de él se comporten correctamente.

Flexbox Utilities


      .flex {
        display: flex;
      }

      .flex-row {
        flex-direction: row;
      }

      .flex-column {
        flex-direction: column;
      }

Las clases de utilidades de flexbox se utilizan para aplicar rápidamente estilos de flexbox a los elementos.

Display Utilities


      .d-none {
        display: none;
      }

      .d-block {
        display: block;
      }

Las clases de utilidades de display se utilizan para controlar rápidamente la propiedad de display de los elementos.
