Interruptor de Tema

El componente de interruptor de tema (ds-theme-switch) permite a los usuarios alternar entre el modo claro y el modo oscuro. Este componente es altamente personalizable y fácil de integrar en cualquier aplicación.

Uso

Ejemplo de uso en HTML:

<ds-theme-switch></ds-theme-switch>

Configuración

El componente de interruptor de tema no requiere propiedades de entrada (@Input). Se integra automáticamente con el servicio de tema para alternar entre el modo claro y el modo oscuro.

Ejemplos de Interruptor de Tema

Accesibilidad

El componente de interruptor de tema incluye soporte para accesibilidad mediante el uso de atributos aria y controles accesibles.

Conclusión

El componente de interruptor de tema es una herramienta útil para mejorar la experiencia del usuario al permitirles elegir entre el modo claro y el modo oscuro. Con su integración sencilla y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
