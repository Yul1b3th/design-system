Temas

En esta sección, exploraremos los temas disponibles en el sistema de diseño. Los temas permiten cambiar la apariencia de la aplicación de manera consistente y fácil.

Tema Oscuro

El tema oscuro aplica un esquema de colores oscuros a la interfaz, proporcionando una apariencia elegante y moderna. 


      body.dark-mode {
        background-color: get-background("inverted");
        color: get-color("white");

        h1,
        h2,
        h3,
        h4,
        h5,
        h6,
        p,
        small,
        strong,
        em,
        blockquote,
        code,
        pre,
        a {
          color: get-color("white");
        }
      }

Estos estilos se aplican cuando el tema oscuro está activado, asegurando que todos los elementos de texto y fondo utilicen colores adecuados para una mejor legibilidad en entornos oscuros.

Tema Claro

El tema claro aplica un esquema de colores claros a la interfaz, proporcionando una apariencia limpia y brillante. 


      body.light-mode {
        background-color: get-background("default");
        color: get-color("gray-dark");

        h1,
        h2,
        h3,
        h4,
        h5,
        h6 {
          color: get-color("dark-200");
        }

        p,
        small,
        strong,
        em,
        blockquote,
        code,
        pre,
        a {
          color: get-color("gray-dark");
        }
      }

      // Ejemplo de estilos adicionales para el modo claro
      .card {
        background-color: get-background("default");
        border-color: get-color("primary-200");
        color: get-color("dark");
      }

Estos estilos se aplican cuando el tema claro está activado, asegurando que todos los elementos de texto y fondo utilicen colores adecuados para una mejor legibilidad en entornos claros.
