Inputs

Los componentes de entrada (ds-input) se utilizan para permitir a los usuarios ingresar datos en la aplicación. Estos componentes son altamente personalizables y soportan múltiples tipos de entradas para adaptarse a diferentes contextos de uso.

Tipos de Inputs

A continuación se presentan los diferentes tipos de componentes de entrada disponibles:

  • Text Input
  • Number Input
  • Email Input
  • Password Input
  • Date Input
  • Select Input
  • Checkbox
  • Radio
  • Textarea
  • Switch

States

Los diferentes estados del componente de entrada de texto muestran cómo se comporta el componente en diversas situaciones.

Default

Hover

El estado hover se muestra cuando el usuario pasa el cursor sobre el campo de entrada.

Typing (Focus)

El estado de enfoque se muestra cuando el usuario hace clic en el campo de entrada y comienza a escribir.

Disabled

Verification

El estado de verificación se muestra cuando el campo de entrada ha sido verificado.

Error

El estado de error se muestra cuando hay un error en el campo de entrada.

Text Input

El componente ds-text-input se utiliza para capturar datos de texto con opciones de personalización y validación. Es completamente accesible y adecuado para formularios o campos de entrada independientes.

Uso

Ejemplo de uso en HTML:


        <form [formGroup]="form">
          <ds-text-input
            formControlName="textInput"
            id="name"
            label="Name"
            placeholder="Enter your name"
            [ariaRequired]="true"
            [maxlength]="50"
            [iconSvg]="textIconSvg"
          ></ds-text-input>
        </form>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
label string Texto mostrado como etiqueta del campo. "Nombre completo"
placeholder string Texto mostrado dentro del campo cuando está vacío. "Escribe tu nombre"
formControlName string Vincula el componente a un control de un formulario reactivo. "name"
required boolean Define si el campo es obligatorio. true
disabled boolean Desactiva el campo de entrada. false
maxlength number Longitud máxima del texto ingresado. 50
errorMessage string Mensaje de error personalizado mostrado para errores de validación. "Este campo es obligatorio"
iconSvg string SVG del icono a mostrar dentro del campo de entrada. <svg ...></svg>

Select Input

El componente ds-select-input se utiliza para permitir a los usuarios seleccionar una opción de una lista desplegable. Este componente es altamente personalizable y soporta múltiples configuraciones.

Uso

Ejemplo de uso en HTML:


        <form [formGroup]="form">
          <ds-select-input
            formControlName="selectInput"
            id="select"
            label="Select"
            placeholder="Select an option"
            [ariaRequired]="true"
            [options]="[
              { label: 'Option 1', value: 1 },
              { label: 'Option 2', value: 2 },
              { label: 'Option 3', value: 3 }
            ]"
          ></ds-select-input>
        </form>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
label string Texto mostrado como etiqueta del campo. "Seleccionar"
placeholder string Texto mostrado dentro del campo cuando no se ha seleccionado ninguna opción. "Selecciona una opción"
formControlName string Vincula el componente a un control de un formulario reactivo. "select"
required boolean Define si el campo es obligatorio. true
disabled boolean Desactiva el campo de entrada. false
errorMessage string Mensaje de error personalizado mostrado para errores de validación. "Este campo es obligatorio"
options Array<{ label: string, value: any }> Lista de opciones que se mostrarán en el menú desplegable. [{ label: 'Option 1', value: 1 }, { label: 'Option 2', value: 2 }]
