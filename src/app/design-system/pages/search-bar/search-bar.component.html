Search Bar

El componente de barra de búsqueda (ds-search-bar) se utiliza para permitir a los usuarios buscar y seleccionar opciones de una lista desplegable. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:


        <form [formGroup]="form">
          <ds-search-bar
            formControlName="searchValue"
            id="searchValue"
            label="Search"
            placeholder="Type to search..."
            [ariaRequired]="true"
            [options]="['Option 1', 'Option 2', 'Option 3']"
          ></ds-search-bar>
        </form>

Configuración

El componente acepta las siguientes propiedades:

Propiedad Tipo Descripción Ejemplo
label string Texto mostrado como etiqueta del campo. "Buscar"
placeholder string Texto mostrado dentro del campo cuando está vacío. "Escribe para buscar..."
formControlName string Vincula el componente a un control de un formulario reactivo. "searchValue"
required boolean Define si el campo es obligatorio. true
disabled boolean Desactiva el campo de entrada. false
errorMessage string Mensaje de error personalizado mostrado para errores de validación. "Este campo es obligatorio"
options string[] Lista de opciones que se mostrarán en el menú desplegable. ["Option 1", "Option 2", "Option 3"]
