Barra de Búsqueda

El componente de barra de búsqueda (ds-search-bar) se utiliza para permitir a los usuarios buscar contenido dentro de la aplicación. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-search-bar [(ngModel)]="searchQuery" (search)="handleSearch($event)"></ds-search-bar>

Configuración

El componente de barra de búsqueda acepta las siguientes propiedades de entrada (@Input) y eventos de salida (@Output):

  • ngModel (string): El valor de la consulta de búsqueda.
  • placeholder (string): Texto que se mostrará cuando el campo esté vacío.
  • disabled (boolean): Indica si la barra de búsqueda está deshabilitada. Valor por defecto: false.
  • search (EventEmitter<string>): Evento que se emite cuando se realiza una búsqueda.

Accesibilidad

El componente de barra de búsqueda incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de barra de búsqueda es una herramienta esencial para permitir a los usuarios buscar contenido dentro de la aplicación. Con sus múltiples opciones de configuración y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
