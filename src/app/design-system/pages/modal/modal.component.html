Modal

El componente modal (ds-modal) se utiliza para mostrar contenido en una ventana emergente sobre la interfaz principal. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-modal
  [title]="'Título del Modal'"
  [show]="isModalOpen"
  [showOverlay]="true"
  (close)="handleClose()"
  (save)="handleSave()"
>
  <div modal-body>
    <p>Contenido del modal...</p>
  </div>
</ds-modal>

Configuración

El componente modal acepta las siguientes propiedades de entrada (@Input) y eventos de salida (@Output):

  • title (string): Título del modal.
  • show (boolean): Indica si el modal está abierto. Valor por defecto: false.
  • showOverlay (boolean): Indica si se muestra el overlay. Valor por defecto: true.
  • close (EventEmitter<void>): Evento que se emite cuando el modal se cierra.
  • save (EventEmitter<void>): Evento que se emite cuando se guarda el contenido del modal.
