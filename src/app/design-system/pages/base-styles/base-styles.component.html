Estilos Base

En esta sección, se exploran los estilos base que se aplican a todo el sistema de diseño. Estos estilos aseguran una apariencia y comportamiento consistentes en toda la aplicación.

Reboot

El archivo _reboot.scss contiene un conjunto de reglas CSS que reinician los estilos predeterminados del navegador para asegurar una base consistente. Estas reglas incluyen:

  • Box sizing rules
  • Prevent font size inflation
  • Remove default margin
  • Remove list styles
  • Set core body defaults
  • Set shorter line heights on headings and interactive elements
  • Balance text wrapping on headings
  • Default styles for unclassed a elements
  • Make images easier to work with
  • Inherit fonts for inputs and buttons
  • Ensure textareas without rows attribute are not tiny
  • Extra scroll margin for anchored elements

Tipografía

El archivo _typography.scss define los estilos tipográficos utilizados en el sistema de diseño. Estos estilos aseguran una apariencia tipográfica consistente y legible en toda la aplicación.

Estilos Generales


      body {
        font-family: $font-family;
        font-size: get-font-size("base");
        line-height: get-line-height("base");
      }

Estos estilos se aplican al cuerpo del documento, estableciendo la fuente, el tamaño de fuente y la altura de línea predeterminados.

Encabezados


      h1 {
        @include typography("5xl", "xl", 700, "md");
      }

      h2 {
        @include typography("4xl", "lg", 700, "md");
      }

      h3 {
        @include typography("3xl", "lg", 600, "md");
      }

      h4 {
        @include typography("2xl", "lg", 600, "md");
      }

      h5 {
        @include typography("xl", "lg", 500, "md");
      }

      h6 {
        @include typography("lg", "lg", 500, "md");
      }

Estos estilos se aplican a los encabezados h1 a h6, estableciendo el tamaño de fuente, la altura de línea, el peso de la fuente y el margen inferior.

Párrafos y Textos


      p:not(:last-child) {
        @include typography("base", "base", 400, "md");
      }

      small {
        @include typography("sm", "base", 400, "sm");
      }

      strong {
        font-weight: 700;
      }

      em {
        font-style: italic;
      }

Estos estilos se aplican a los párrafos y otros elementos de texto, asegurando una apariencia consistente y legible.

Bloques de Cita


      blockquote {
        font-size: get-font-size("lg");
        line-height: get-line-height("lg");
        margin: get-spacing("md") 0;
        padding: get-spacing("md");
        border-left: 4px solid get-color("primary");
        background-color: get-background("default");
      }

Estos estilos se aplican a los bloques de cita, asegurando una apariencia destacada y legible.

Código y Preformateado


      code,
      pre {
        font-family: "Courier New", Courier, monospace;
        font-size: get-font-size("sm");
        background-color: get-background("default");
        padding: get-spacing("xs");
        border-radius: 4px;
      }

      pre {
        overflow-x: auto;
      }

Estos estilos se aplican a los elementos de código y preformateado, asegurando una apariencia consistente y legible.

Enlaces


      a {
        color: get-color("primary");
        text-decoration: none;

        &:hover {
          text-decoration: underline;
        }
      }

Estos estilos se aplican a los enlaces, asegurando una apariencia consistente y legible.
