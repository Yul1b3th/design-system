Breadcrumb

El componente de breadcrumb (ds-breadcrumb) se utiliza para mostrar la ruta de navegación actual del usuario dentro de la aplicación. Este componente es altamente personalizable y soporta múltiples configuraciones para adaptarse a diferentes contextos de uso.

Uso

Ejemplo de uso en HTML:

<ds-breadcrumb
  [items]="[
    { label: 'Home', url: '#' },
    { label: 'Library', url: '#' },
    { label: 'Data' }
  ]"
></ds-breadcrumb>

Configuración

El componente de breadcrumb acepta las siguientes propiedades de entrada (@Input):

  • items (Array<{ label: string, url?: string }>): Lista de elementos del breadcrumb. Cada elemento puede tener una etiqueta (label) y una URL opcional (url).

Accesibilidad

El componente de breadcrumb incluye soporte para accesibilidad mediante el uso de atributos aria y roles adecuados.

Conclusión

El componente de breadcrumb es una herramienta útil para mejorar la navegación del usuario al mostrar la ruta actual dentro de la aplicación. Con su integración sencilla y soporte para accesibilidad, es una adición valiosa para cualquier aplicación.
